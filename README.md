Health & Lifestyle Data Analysis
Project Overview
This project provides a comprehensive analysis of health and lifestyle data, focusing on fitness, sleep, diet, medication adherence and lifestyle factors. The goal is to uncover patterns, correlations and insights that support healthier living through data-driven analysis.

How to Use This Project
1.	Create the datasets
python create_health_datasets.py
2.	Run the analysis
python analyze_health.py

Key Features
1. Fitness Tracker Analysis
•	Recognizes exercise patterns by day of week and activity type
•	Tracks progress using metrics such as steps, duration and VO2 max
•	Performs correlation analysis between fitness metrics
•	Visualizes exercise frequency and intensity patterns
2. Sleep Quality Analysis
•	Tracks sleep efficiency and duration
•	Analyzes factors affecting sleep (caffeine, stress, screen time)
•	Examines sleep stage composition
•	Compares weekend vs weekday sleep patterns
3. Dietary Habits Analysis
•	Tracks nutritional intake by food group
•	Compares diet types (vegetarian, high-protein, etc.)
•	Analyzes correlations between diet and health outcomes
•	Studies meal timing effects on health
4. Medication Adherence Analysis
•	Calculates adherence rates by medication type
•	Tracks effectiveness over time
•	Identifies day-of-week adherence patterns
•	Correlates adherence levels with health outcomes
5. Lifestyle Survey Analysis
•	Analyzes demographic and lifestyle factors
•	Studies correlations with wellness indicators
•	Clusters lifestyle patterns for insight generation
•	Identifies risk factors for chronic conditions

Project Outputs
•	Five comprehensive PNG visualizations
•	Summary report in text format
•	All datasets in CSV format
•	Statistical analysis results, including correlation matrices and cluster profiles
